# Simple Unity Game Prototype

Project structure:
- Core: movement, collectibles, score, independent gameplay loop
- Extent: burst skill activated by Space, independent feature

How to run:
1. Open the project in Unity.
2. Enter Play mode.
3. Move with WASD or arrow keys.
4. Press Space to trigger the burst ability.

Notes:
- Core and Extent are implemented as separate modules.
- If either module is missing, the remaining gameplay still works.
